Welcome to the ACME University Admin Panel

Use this dashboard to manage users, courses, and enrollments throughout the system. All admin tools are available in the navigation bar above.

Admin Navigation Guide

  • User: Create, edit, or delete Students, Teachers, and Admin accounts.
  • Course: Add new courses, assign teachers, and configure course capacity and meeting times.
  • Enrollment: View all enrollments. See student usernames, full names, courses, and grades.

Admin Capabilities (Per Project Requirements)

  • Create new user accounts (student, teacher, admin).
  • Assign teachers to courses using a dropdown selector.
  • View detailed enrollment information for every student.
  • Edit or update grades for each student enrollment.
  • Delete or modify any record in the system as required.

If you need help at any time, navigate using the top menu or revisit this page for guidance.
